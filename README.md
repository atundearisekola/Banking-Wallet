# Banking-Wallet


# Running our application services with Docker
### To start this app run
 docker-compose up --build -d

 #### Api Link

 * /api/wallet/add-wallet   To add Wallet

 * /api/wallet/all-wallet   To view wallet

 * /api/wallet/send-fund   To send fund to other wallet

 * /api/users/currentuser   Get user wallet and transaction history

 * /api/users/all-users     Get all user in the database

 * /api/wallet/:id           Get wallet details
